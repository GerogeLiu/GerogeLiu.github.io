如何搭建自己的个人博客

简介

这个博客是使用hexo博客框架搭建的，在视频的指引下，很快就拥有了这个属于自己的博客。

什么是Hexo

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

查看文档

搭建步骤

下面记录下自己这次的搭建过程：

  1. 由于hexo框架是基于node的，所以先下载并安装nodejs

  2. 安装nodejs是自带包管理器npm的，但是由于国内的特殊关系，我们需要借助cnpm来帮忙。

  3. 使用cnpm 安装 hexo-cli

  4. 新建文件夹blog, 切换到该目录下

  5. 使用hexo命令初始化，自动下载需要的框架文件和默认的主题landscape

  6. 本地开启服务查看博客，启动4000端口

  7. 浏览器地址栏输入http://localhost:4000即可看到搭建起来的博客

hexo的常用命令

  1. 运行服务命令
    $ hexo server
    或
    $ hexo s
  2. 新建文章命令
    $ hexo new 'new post'
    或
    $ hexo n 'new post'
  3. 删除数据库命令
    $ hexo clean
  4. 生成静态文件命令
    $ hexo generate
    或
    $ hexo g
  5. 部署命令
    $ hexo deploy

部署到github上

由于github提供免费的静态网页展示的功能，我们可以将博客放到github上，其他人就可以通过github.io来访问我们的博客啦。

步骤：

  1. 登陆github，并新建一个仓库，取名为.github.io
  2. 安装hexo部署到git上的插件
  3. 打开_config.yml文件写入需要的配置信息
  4. 部署到github

    主题要怎么更换呢?

  5. 从github上下载需要的主题如yilia
  6. 修改_config.yml文件
  7. 先使用清除命令，后使用生成命令
  8. 重新推送到github
  9. 搞定
